# RevealScrim

A scrim that reveals the content as the user scrolls. Can be used as preChildren for ScrollyElement.

## Usage

Note: There should be a `ScrollyElement` higher up in the tree for `RevealScrim` to work.

[View Story](?path=/story/components-revealscrim--default)

```tsx
import { ScrollyProvider } from "react-scrolly-telling";
import ScrollyElement from "react-scrolly-telling/element";
import RevealScrim from "react-scrolly-telling/scrim";

function App() {
  return (
    <ScrollyProvider>
      <ScrollyElement preChildren={<RevealScrim />}>{children}</ScrollyElement>
    </ScrollyProvider>
  );
}
```

## API

The component accepts the following props:

- `fromOpacity`: The starting opacity of the background-color when content is entering or exiting the screnn. Defaults to 1.
- `toOpacity`: The final opacity of the background-color when content is in completely visible. Defaults to 0.
- `backgroundColor`: The background-color of the scrim. Defaults to `black`.
- Any props that are accepted by `div`..
